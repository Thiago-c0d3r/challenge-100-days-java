### forma q vimos:
qnd vc cria arrays vc define tamanhos fixos

`int[] numeros = new int[5];`

nesse exemplo acima, vc definiu um array de inteiros com 5 espaços

### Problema
imagine que vc precisa de um sistema de pedidos:
vc n sabe quantos pedidos, ou seja "como definir o tamanho do array?"
a resposta: n tem como, qnd vc n sabe fica mt imprevisível

### Solução
ArrayList: basicamente um array que cresce sozinho

### Analogia
pense numa mochila com espaços infinitos, quando vc coloca coisas dentro dela ela vai aumentando de tamanho automaticamente
